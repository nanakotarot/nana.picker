# ✨ 幸運抽籤與分組神器 (Lucky Draw & Grouping Tool)

## 📖 專案簡介
這是一個現代化、極簡風格的網頁工具，專為需要快速進行名單抽籤或隨機分組的場合設計。無論是活動抽獎、課堂分組還是團康遊戲，都能輕鬆應對。無需安裝任何軟體，單一檔案即可運作。

## 🚀 功能特色
- **完全響應式設計 (RWD)**：從手機到桌機都能完美顯示，介面自動適應。
- **雙模式切換**：
  - 🎲 **幸運抽籤**：指定抽出人數，隨機選取幸運兒。
  - 👥 **隨機分組**：支援「按組數分」或「按每組人數分」兩種邏輯。
- **極致視覺體驗**：採用 Glassmorphism (毛玻璃) 風格，搭配流暢動畫與深色模式支援。
- **純前端運作**：所有資料皆在瀏覽器端處理，隱私安全且無須網路也能使用（字體除外）。

## 🛠 使用方法
1. 下載本專案中的 `index.html` 檔案。
2. 直接雙擊檔案，或將其拖入任何現代瀏覽器 (Chrome, Safari, Edge 等) 開啟。
3. 在左側輸入名單（一行一個名字）。
4. 設定您需要的模式（抽籤或分組）與參數。
5. 點擊開始按鈕，觀看結果！

## ⚙️ 自定義指南
本專案採用單一 HTML 結構，方便修改與部署。

### 代碼結構
- **樣式 (CSS)**：位於 `<style>` 標籤內。使用了 CSS Variables 定義配色（如 `--primary-gradient`），方便快速換色。
- **結構 (HTML)**：
  - `.container`：主容器，使用 Grid 佈局處理 RWD。
  - `.input-section`：左側（或行動版上方）的控制面板。
  - `.results-container`：右側（或行動版下方）的結果展示區。
- **邏輯 (JS)**：位於 `<script>` 標籤內。
  - `switchMode(mode)`：切換抽籤/分組模式。
  - `performDraw(names)`：執行抽籤邏輯。
  - `performGrouping(names)`：執行分組邏輯。

若要修改預設名單，可搜尋代碼中的 `<textarea id="nameInput">` 並修改其內容。
