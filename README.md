# 幸運抽籤與分組神器 - 賽博龐克版
# Lucky Draw & Grouping Tool - Cyberpunk Edition

![AI Powered](https://img.shields.io/badge/AI-Powered-purple?style=flat-square&logo=google-gemini&logoColor=white)

這是一個現代化、極具未來感的網頁工具，專為需要快速進行名單抽籤或隨機分組的場合設計。全新升級為 **Cyberpunk Neon Glassmorphism** 風格，結合深邃的星空背景與動態霓虹光效，為您的活動增添強烈的視覺衝擊。

This is a modern, futuristic web tool designed for quick list drawing or random grouping. Upgraded with a **Cyberpunk Neon Glassmorphism** style, it combines a deep starry background with dynamic floating light orbs to add a strong visual impact to your events.

> **⚠️ 注意 (Note)**: 本專案由 AI 協助製作、修改與維護。
> This project is created, modified, and maintained by AI.

## 功能特色 (Features)

- **Cyberpunk 未來風格 (Cyberpunk Future Style)**：
    - 深色星空背景、動態漂浮光球、高對比的霓虹配色，打造沉浸式體驗。
      Deep starry background, dynamic floating light orbs, and high-contrast neon colors create an immersive experience.

- **玻璃擬態 (Glassmorphism)**：
    - 全介面採用半透明磨砂玻璃質感，搭配發光邊框，科技感十足。
      The interface features a semi-transparent frosted glass texture with glowing borders, providing a high-tech feel.

- **完全響應式設計 (Fully Responsive Design)**：
    - 重新設計的穩固佈局，從手機到 4K 螢幕都能完美顯示。
      Redesigned robust layout that displays perfectly from mobile phones to 4K screens.

- **雙核心功能 (Dual Core Functions)**：
    - **幸運抽籤 (Lucky Draw)**：大螢幕等級的抽獎動畫，支援單人及多人同時抽出，結果具有震撼的發光特效。
      Big-screen level lucky draw animation, supporting single or multiple winners with stunning glowing effects.
    - **隨機分組 (Random Grouping)**：支援「按組數分」或「按每組人數分」，結果卡片清晰易讀。
      Supports grouping by "Number of Groups" or "People per Group", with clear and readable result cards.

- **純前端運作 (Pure Frontend)**：
    - 所有資料與邏輯皆在瀏覽器端極速處理，確保隱私。
      All data and logic are processed locally in the browser, ensuring privacy and speed.

## 技術堆疊 (Tech Stack)

- **HTML5**: 語意化結構 (Semantic structure).
- **CSS3**:
    - 使用 `:root` 變數管理霓虹色系。
      Uses `:root` variables for neon color management.
    - 大量運用 `backdrop-filter` 實現玻璃特效。
      Extensive use of `backdrop-filter` for glass effects.
    - CSS Animation 實現流暢的進場與互動動畫。
      CSS Animation for smooth entrance and interaction effects.
- **JavaScript (ES6+)**:
    - 模組化邏輯，包含陣列洗牌算法 (Fisher-Yates Shuffle) 與 Canvas 紙花特效。
      Modular logic, including Fisher-Yates Shuffle algorithm and Canvas confetti effects.

## 專案結構 (Project Structure)

- `style.css`: 視覺核心，定義了所有的 Cyberpunk 配色、玻璃材質與 RWD 斷點 (Visual core defining all Cyberpunk colors, glass textures, and RWD breakpoints).
- `script.js`: 大腦中樞，處理資料流、抽籤邏輯與 Canvas 紙花繪製 (Brain center handling data flow, drawing logic, and Canvas confetti rendering).
